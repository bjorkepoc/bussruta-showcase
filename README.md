# Bussruta

Bussruta is a digital version of a social card game built for groups playing together in the same room.

This repository is a public showcase only. The application source code is kept in a private repository.

## What It Does

- Supports local play on one shared device.
- Supports hosted play where one device runs the table and other players join from their own devices.
- Includes warmup rounds, pyramid gameplay, tie-breaks, and the final bus route challenge.
- Keeps the experience focused on quick setup, clear turn flow, and group-friendly pacing.

## Current Status

Bussruta is under active private development. The core game flow and hosted multiplayer direction are implemented in the private app repository, with manual device QA and release-readiness work still ongoing.
