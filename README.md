# Bussruta Showcase

Public showcase site for Bussruta, a private-source social card game app.

The showcase is intentionally static and contains only public-facing product copy, app screenshots, and presentation assets. It does not include application source code, internal implementation notes, tests, or private development documentation.

## Local Preview

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## Public Site

This repository is configured for GitHub Pages through `.github/workflows/pages.yml`.

## Source Code

The Bussruta application source code is private.
